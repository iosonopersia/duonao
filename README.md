# duonao
  1. Get the needed tools:
```bash
cd duonao
pip install -U pip setuptools
pip install virtualenv
```
  2. Prepare the virtual environment:
```bash
python -m venv venv
```
  3. Install the dependencies:
```bash
venv\Scripts\activate
pip install -r requirements.txt
deactivate
```
  4. Run the script:
``` bash
python main
```
